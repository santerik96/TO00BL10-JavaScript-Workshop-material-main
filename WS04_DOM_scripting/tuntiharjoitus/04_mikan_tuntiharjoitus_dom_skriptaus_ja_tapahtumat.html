DOM-skriptaus Workshop Harjoitus

Liitetään Visual Studio Code Githubiin, ja synkataan harjoitus sinne. Ope demoaa.

Vaihe 1: Perustiedot ja valmistelut

Luo uusi HTML-tiedosto ja lisää perusrakenne:

<!DOCTYPE html>
<html lang="fi">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>DOM-skriptaus Harjoitus</title>
</head>
<body>
    <div id="content"></div>
    <form id="userForm">
        <input type="text" id="userInput" placeholder="Kirjoita jotain...">
        <button type="submit">Lisää</button>
    </form>
    <form id="extendedForm">
        <input type="text" id="firstName" placeholder="Etunimi">
        <input type="text" id="lastName" placeholder="Sukunimi">
        <select id="dropdown">
            <option value="option1">Vaihtoehto 1</option>
            <option value="option2">Vaihtoehto 2</option>
            <option value="option3">Vaihtoehto 3</option>
        </select>
        <button type="button" id="submitButton">Lähetä</button>
    </form>
    <table id="dataTable">
        <thead>
            <tr>
                <th>Etunimi</th>
                <th>Sukunimi</th>
                <th>Valinta</th>
            </tr>
        </thead>
        <tbody>
        </tbody>
    </table>
    <script src="script.js"></script>
</body>
</html>

Luo uusi JavaScript-tiedosto nimeltä script.js ja varmista, että se on linkitetty HTML-tiedostoon.

Vaihe 2: Sisällön lisääminen innerHTML-attribuutilla

Avaa script.js ja lisää seuraava koodi:

const contentDiv = document.getElementById('content');

contentDiv.innerHTML = `
    <h1>Tervetuloa DOM-skriptauksen harjoitukseen!</h1>
    <p>Tämä sisältö on lisätty käyttäen innerHTML-ominaisuutta.</p>
`;

Vaihe 3: Sisällön lisääminen createElement-funktiolla

Lisää script.js-tiedostoon seuraava koodi:

const newElement = document.createElement('div');

newElement.innerHTML = `
    <h2>Lisätty elementti</h2>
    <p>Tämä sisältö on lisätty käyttäen createElement-funktiota ja innerHTML-ominaisuutta.</p>
`;

contentDiv.appendChild(newElement);

Vaihe 4: Yhdistä molemmat menetelmät

Lisää script.js-tiedostoon seuraava koodi:

const anotherElement = document.createElement('div');

const header = document.createElement('h3');
header.textContent = 'Toinen lisätty elementti';

const paragraph = document.createElement('p');
paragraph.textContent = 'Tämä sisältö on lisätty käyttäen createElement-funktiota ja textContent-ominaisuutta.';

anotherElement.appendChild(header);
anotherElement.appendChild(paragraph);

contentDiv.appendChild(anotherElement);

Vaihe 5: Käyttäjän syötteen käsittely

Lisää script.js-tiedostoon koodi, joka käsittelee lomakkeen syötteen: 

const form = document.getElementById('userForm');
form.addEventListener('submit', function(event) {
    event.preventDefault();
    const userInput = document.getElementById('userInput').value;

    const userElement = document.createElement('p');
    userElement.textContent = userInput;

    contentDiv.appendChild(userElement);
});

Vaihe 7: Elementtien poistaminen

Lisää nappi elementtien poistamiseen:

const removeButton = document.createElement('button');
removeButton.textContent = 'Poista sisältö';
contentDiv.appendChild(removeButton);

removeButton.addEventListener('click', function() {
    contentDiv.innerHTML = '';
});

Vaihe 8: Dynaaminen sisällön luominen

Luo dynaamisesti lista käyttäjän syötteistä:

const list = document.createElement('ul');
contentDiv.appendChild(list);

form.addEventListener('submit', function(event) {
    event.preventDefault();
    const userInput = document.getElementById('userInput').value;

    const listItem = document.createElement('li');
    listItem.textContent = userInput;

    list.appendChild(listItem);
});

Vaihe 9: Lomakkeen luominen ja kuuntelijan lisääminen

Lisää script.js-tiedostoon koodi, joka käsittelee uuden lomakkeen syötteen: 

const submitButton = document.getElementById('submitButton');
submitButton.addEventListener('click', function() {
    const firstName = document.getElementById('firstName').value;
    const lastName = document.getElementById('lastName').value;
    const dropdown = document.getElementById('dropdown').value;

    console.log(`Etunimi: ${firstName}, Sukunimi: ${lastName}, Valinta: ${dropdown}`);
});

Vaihe 10: Lomakkeen tietojen lisääminen taulukkoon

Lisää script.js-tiedostoon koodi, joka lisää lomakkeen tiedot taulukkoon:

const dataTable = document.getElementById('dataTable').getElementsByTagName('tbody')[0];

submitButton.addEventListener('click', function() {
    const firstName = document.getElementById('firstName').value;
    const lastName = document.getElementById('lastName').value;
    const dropdown = document.getElementById('dropdown').value;

    const newRow = dataTable.insertRow();

    const cell1 = newRow.insertCell(0);
    const cell2 = newRow.insertCell(1);
    const cell3 = newRow.insertCell(2);

    cell1.textContent = firstName;
    cell2.textContent = lastName;
    cell3.textContent = dropdown;
});
